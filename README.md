# 🌤️ Skyline Weather App

A modern, fast, and responsive weather web application built using clean Vanilla JavaScript. It fetches live meteorological data from WeatherAPI to provide users with up-to-date weather information for any city worldwide.

## Features
- **Real-time Data:** Fetches current temperature, humidity, wind speed, and weather conditions instantly.
- **Smart Search (Debouncing):** Optimized search inputs that wait 500ms after the user stops typing before sending API requests, significantly reducing server load.
- **Responsive UI:** Clean design that looks great on mobile, tablet, and desktop screens.
- **Secure Architecture:** Built with a decoupled design to keep private API keys protected from being exposed in public repositories.
